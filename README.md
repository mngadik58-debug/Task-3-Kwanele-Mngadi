# Task-3-Kwanele-Mngadi
* SQL Data Analysis

Welcome to Project 3: SQL Data Analysis. This project forms part of the Industrial Training Kit by DecodeLabs and focuses on mastering the core principles of SQL for data extraction and analysis.

Unlike spreadsheet-based analysis, this project emphasizes the use of structured SQL queries to transform raw transactional data into actionable business insights. The objective is to develop a strong understanding of relational databases and the logical execution of SQL queries.

By completing this project, I demonstrate my ability to bridge the gap between large datasets and meaningful business decisions using pure relational logic.
